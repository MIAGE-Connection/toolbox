Générateur de base de remboursement

Ce petit outil permet de généner la base de remboursement pour les événements de MIAGE Connection.

Pour pouvoir utiliser cet outil, il faut que vous aillez un compte API sur le site https://api.viamichelin.fr. Après avoir créer votre compte vous allez recevoir un mail avec à l'interieur une clé qui ressemble à JSV2GP20220427234158912459843975$165380.
